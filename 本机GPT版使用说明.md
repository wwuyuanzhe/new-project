# 本机 GPT 版家庭小助手使用说明

## 第一次使用

1. 双击 `设置GPT密钥.cmd`
2. 粘贴你的 OpenAI API Key
3. 双击 `启动GPT版小助手.cmd`
4. 浏览器会打开 `http://localhost:3000`

只有通过 `http://localhost:3000` 打开的页面，才会调用 GPT。

如果直接打开 `index.html`，或者打开 GitHub Pages 地址，页面仍然可以用，但一句话识别会使用本地规则，不会调用 GPT。

## 每次使用

双击 `启动GPT版小助手.cmd`，保持黑色窗口不要关闭。

## 给家里其他人用

在同一个家里 Wi-Fi 下，可以先用妈妈这台电脑运行后端。下一步如果要让其他手机访问，需要把地址从 `localhost` 改成这台电脑的局域网 IP，例如：

`http://192.168.1.23:3000`

这一步可能需要开放 Windows 防火墙端口。建议等本机先跑稳定后再做。

## 安全提醒

`.env` 文件里保存的是你的 OpenAI API Key，不要发给别人，也不要上传到 GitHub。
